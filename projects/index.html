CAVEMAN

CAVEMAN

Cave Autonomous Vehicle for Exploration, Mapping, and Navigation


Project Summary:

Led a team of 5 engineers to design and create an autonomous mapping robot for caves/mines to mitigate dangers, built completely from scratch in 4 months.
Caves are inherently dangerous, posing risk to civilians, first responders, explorers, etc. and current cave maps are extremely outdated. We strived to create a solution to all of these problems with CAVEMAN. This rover autonomously navigates caves, maneuvering through the unpredictable terrain, taking pictures of the environment so that it can be turned into a map using our mapping pipeline.

  • Researched the problem with caves/mines
  • Met with multiple robotics and mining professionals
  • Created 2 prototypes of the working rover fully from scratch.
  • Interfaced with 13 sensors & actuators
  • Created low level movement software
  • Created high level navigation software
  • Designed custom PCB motherboard
  • Designed Image-to-Map Pipeline
  • Presented work at Design Expo

Project Achievements:

  • Called the "Best Project I've ever seen" by Department Head & Won 3rd Place at Design Expo
  • Created 10 colored maps of explored & documented environments with high accuracy viewable on the web or in virtual reality
  • Designed image dataset pruning algorithm to reduce computation time & increase accuracy of Photogrammetry pipeline
  • Created custom Communication Protocol for Rover
  • Created extensible Camera Movement Framework using 2 servos, ROS2, and XML
  • Created custom website for map-viewing and documentation
  • Managed all aspects of project from hardware considerations, software contributions & development, sensor interfacing, testing, and presentation
